# SPICEBridge

AI-powered circuit design through simulation.

**Status:** Early development

## Requirements

- Python >= 3.10
- [ngspice](https://ngspice.sourceforge.io/) installed and on PATH

## Install

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -e ".[dev]"
```
